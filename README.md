Otopark Yönetim Simülasyonu (C++)

PROJE HAKKINDA:

Bu proje, C++ kullanılarak geliştirilmiş bir otopark yönetim simülasyonudur. Araç giriş-çıkış işlemleri ve otopark kapasite yönetimi simüle edilmiştir.

ÖZELLİKLER:

-Araç giriş ve çıkış işlemleri
-Otopark kapasite takibi
-Durum görüntüleme (boş/dolu alanlar)
-Basit kullanıcı etkileşimi (konsol üzerinden)

KULLANILAN TEKNOLOJİLER:

-C++
-Standart kütüphaneler

AMAÇ:

Bu proje ile C++ dilinde algoritma geliştirme, veri yönetimi ve simülasyon mantığı üzerine deneyim kazanılmıştır.
